--------------------------------------------------

Access Modifier
1). public		- within class, with in the package, outside of the package, everywhere. (public variable, public method, public class)
2). protected	- protected variable and methods can be in another class in same package only through inheritance. protected variable and methods can be used in another package only through inheritance.
3). default		- we can access class, variable and method within the package (class Test{}, void display(), int a)
4). private		- we can access variable and method within the class (private variable, private method)
